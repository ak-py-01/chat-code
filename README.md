# Chat Code

Turn AI-generated markdown into a real project directory.

## Demo GIF (WIP)

## Why?

ChatGPT and Claude often generate complete apps, but copying each file manually is painful.

## Features

- Paste markdown
- Detect file names and code blocks
- Build folder structure
- Download ZIP
- Runs locally in browser
- Open source
